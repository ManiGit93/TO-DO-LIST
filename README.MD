# Smart To-Do List Application

A full-stack To-Do web application built using Python and Flask.

## Features
- Add / Delete Tasks
- User Authentication
- SQLite Database
- Daily Motivational Quotes (Web Scraping)
- Task Automation
- Responsive UI (Bootstrap)

## Technologies
- Python
- Flask
- SQLite
- BeautifulSoup
- Requests
- Schedule
- Git & GitHub

## Run
pip install -r requirements.txt
python app.py
